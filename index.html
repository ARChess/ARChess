    Welcome.

    ARC (Augmented Reality Chess) is an experimental project to create a game of chess in an augmented reality. Why? Well, for starters, we needed a project for our class, but that isn't the only reason. Think of the cool side effects of virtualizing the game of chess. You could receive feedback on the game your playing as your playing it. You could also be alerted to illegal moves or check/mate. Finally, you can play a game of chess anywhere that you have a data connection, yet it feels like your playing on an authentic board. This should not only serve to help make chess easier to learn but also make chess more fun and interactive drawing in a new generation of players.

    Awesome, Now What?

    ARC is still in development, but that doesn't mean you can't check it out in the mean time. You'll need a Windows Phone, in development mode, running at least 7.1 and a computer with Visual Studio 2010 Express installed. Download a copy of the source using the links at the top of the page and run the solution file inside the download.

    Authors and Contributors

    Support or Contact

    ARC is still in the early experimental phases, meaning we will only offer limited support. If you run into a bug, please feel free to report it. We'll fix the bug in the next release.
